Questions:
- What happens if users choose "Cancel"?

Trong lớp JOptionPane, mỗi nút bấm tương ứng với một giá trị nguyên (int) cụ thể:
YES_OPTION: 0
NO_OPTION: 1
CANCEL_OPTION: 2
CLOSED_OPTION (khi chọn dấu X): -1

Trong đoạn code, biểu thức điều kiện: (option == JOptionPane.YES_OPTION ? "Yes" : "No") chỉ để kiểm tra xem bạn có chọn Yes hay không
Nếu chọn Cancel (giá trị = 2), điều kiện 2 == 0 là sai
Kết quả chương trình sẽ hiển thị: "You've chosen: No"


- How to customize the options to users, e.g. only two options: "Yes" and "No", OR "I do" and "I don't" (Suggestion: Use Javadocs or using Eclipse/Netbean IDE help)

Để tuỳ chỉnh các nút bấm có 2 cách tiếp cận dựa trên tài liệu Javadocs của JOptionPane:

a) Chỉ hiện thị 2 nút "Yes" và "No"
Thêm tham số optionType vào hàm showConfirmDialog:
	int option = JOptionPane.showConfirmDialog(null,
				"Do you want to change to the first class ticket?",
				"Select an Option",
				JOptionPane.YES_NO_OPTION);

b) Tuỳ chỉnh nội dung chữ trên nút ("I do" và "I don't")
Tạo phiên bản đầu đủ tham số của showOptionDialog:
	Object[] options = {"I do", "I don't"};
	int n = JOptionPane.showOptionDialog(null,
			"Do you want to change to the first class ticket?",
			"A Silly Question",
			JOptionPane.YES_NO_OPTION,
			JOptionPane.QUESTIN_MESSAGE,
			null, options, options[0]);
	String choice = (n == 0) ? "I do" : "I don't";
	JOptionPane.showMessageDialog(null, "You've chosen: " + choice);
